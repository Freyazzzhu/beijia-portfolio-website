Visual Direction:
Create a clean, soft, professional vertical scrolling section. Each skill panel should feel like a separate calm page or card, with generous spacing, elegant typography, and subtle visual details. Use soft light colours, minimal icons, and gentle transitions. Avoid percentage bars or exaggerated skill ratings.

Layout Suggestion:
Each panel can occupy most of the screen height, allowing users to scroll vertically through four skill categories. Use a consistent layout for all panels: category title, short description, and skill list. The design should feel polished, readable, and slightly dreamy, matching the overall portfolio style.
